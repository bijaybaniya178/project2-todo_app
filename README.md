# project2-todo_app

Python Todo App using PostgreSQL  A simple command-line Todo application where users can add, view, update, and delete tasks stored in a PostgreSQL database. Uses raw SQL queries via psycopg2 for database operations.

## Setup

1. Clone the repo
2. Create a .env file in the project root with the following keys:
   DB_HOST, DB_NAME, DB_USER, DB_PASSWORD, DB_PORT
